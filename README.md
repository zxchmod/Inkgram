# 🌌 Inkgram

**Inkgram** is a third-party messaging app powered by the **Telegram API**.  
It delivers the full Telegram experience with extra features that enhance usability, style, and convenience.

**Status:** In the development 🚧

---

## ✨ Features

- ✅ **All core Telegram features** — secure, fast, and reliable messaging.  
- 🌍 **Built-in translator** — translate messages instantly using multiple translation engines.  
- 🎨 **Customizable themes** — switch between different styles, including **Dark Ink** and more.  
- ⚡ **Lightweight & fast** — optimized performance with smooth animations.  
- 🔒 **Privacy friendly** — respects Telegram’s secure infrastructure.  

---

## 📦 Installation

### Android
1. Clone this repository:
   ```bash
   git clone https://github.com/zxchmod/Inkgram.git
   ```
2. Open the project in **Android Studio**.  
3. Build and run on a device or emulator.  

*(Note: This project is not affiliated with Telegram Messenger LLP. It only uses the official Telegram API.)*

---

## 🛠 Tech Stack

- **Kotlin / Java** — core app code.  
- **Telegram API** — messaging and communication.  
- **Gradle** — build system.
